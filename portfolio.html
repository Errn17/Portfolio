Beer-Palate

This is the first full project in my portfolio. We created a website where you can search for new beers using Punk API. We added an age verification to make sure you are of age to enter the site, otherwise you are redirected to a kid friendly site. You can also search for a brew and if you find one you like add it to your favorites page to save for later! Also, we added a link to LYFT to prevent drinking and driving. We plan to add more features like finding local bars and utilizing more beer apis to expand the search results! Feel free to hit the link below to check the site out for yourself!
